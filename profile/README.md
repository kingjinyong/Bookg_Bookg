## ğŸš€ How to Start
### Clone Repository 
```
$ git clone --recursive https://github.com/A-Interview/Docker.git
```
### env setting in the Settings folder 
* settings/.env
```
# backend env
GPT_API_KEY=

DJANGO_SECRET_KEY=
MYSQL_DATABASE=
MYSQL_NAME=
MYSQL_USER=
MYSQL_PASSWORD=

```
$docker compose up --build
